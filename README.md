# Time Series Analysis of Births in Albania

## Description
This project analyzes birth data in Albania using time series methods on R programming language.
The goal is to study trends, seasonality, and patterns over time.

## Data
- Birth statistics in Albania (time series data)

## Methods
- Time series visualization
- Trend and seasonality analysis
- Decomposition
- AR / MA / ARIMA models
- Forecasting

## Tools & Libraries
- R
- tidyverse
- ggplot2
- forecast
- stats

## Files
- analysis.Rmd – full analysis and code
- analysis.html – rendered report
- births_albania.csv – dataset
